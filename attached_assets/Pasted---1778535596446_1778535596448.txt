דחוף — צריך להחזיר את אזור “צור קשר / שליחת הזמנה” למבנה התקין שלו.

הבעיה:
בעקבות השינוי האחרון, כל טופס שליחת ההזמנה נכנס לתוך אזור הסיום של האתר, וזה שבר את העמוד.
כרגע באזור “רוצים להזמין טעם שאהבתם?” מופיעים שדות טופס מלאים:
שם מלא
טלפון
הערות / פרטי הזמנה

זה לא צריך להיות שם.

הדרישה המרכזית:
אזור הסיום באתר צריך להישאר אזור צור קשר נקי בלבד, עם:
כותרת
טקסט קצר
טלפון
מייל
כפתור “שליחת הזמנה”
חתימה “באהבה, ליאור”

הטופס המלא של ההזמנה צריך להישאר רק בתוך חלונית ההזמנה / modal / סל ההזמנה, ולא להיות מוצג בתוך הדף עצמו.

מה לתקן:

1. להסיר מאזור contact-section / final / contact את הטופס המלא
אין להציג בתוך אזור הסיום:
input של שם מלא
input של טלפון
textarea של הערות
שדות חובה
טופס הזמנה מלא

2. להחזיר את אזור הסיום למבנה המקורי
המבנה הרצוי באזור הסיום:

<section class="final site-section contact-section" id="contact">
  <div class="container">
    <div class="reveal">
      <div class="section-label">- בואו נדבר -</div>
      <h2>רוצים להזמין טעם שאהבתם?</h2>
      <p>ספרו לליאור איזה טעם אהבתם ונמשיך משם בצורה פשוטה ונעימה.</p>

      <div class="contact-details-card">
        טלפון
        מייל
      </div>

      <a class="btn contact-submit" href="#" data-contact-whatsapp target="_blank" rel="noopener noreferrer">
        שליחת הזמנה
      </a>

      <div class="signature">באהבה, ליאור</div>
    </div>
  </div>
</section>

3. לוודא שכפתור “שליחת הזמנה” באזור הסיום לא פותח טופס בתוך הדף
הכפתור צריך:
- או לפתוח WhatsApp ישירות
- או לפתוח את modal ההזמנה הקיים
אבל בשום מצב לא להכניס את שדות הטופס לתוך אזור הסיום.

4. להשאיר את טופס ההזמנה רק בתוך orderModal
הטופס הבא צריך להישאר רק בתוך:
<div class="order-modal" id="orderModal">

כל השדות:
customerName
customerPhone
manualOrderField / orderProduct
orderDate
orderNotes
orderForm

צריכים להיות רק בתוך ה-modal, לא בתוך contact-section.

5. לבדוק שאין CSS שהפך את orderForm לגלוי בתוך הדף
בדוק אם נוספו כללים כמו:
.order-form { display: block; }
.order-details-step { display: block; }
form { display: block; }

אם כן — לא לשבור את hidden.
הכלל התקין:
אלמנט עם hidden חייב להישאר מוסתר:
[hidden] {
  display: none !important;
}

6. לתקן את גודל כרטיס צור קשר בלי להכניס טופס
הכרטיס צריך להיות קומפקטי ונקי, אבל רק עם פרטי קשר.

- הכרטיס יהיה ברוחב כמעט מלא
- לא להציג טופס בתוך אזור הסיום
- רק פרטי קשר וכפתור

בדיקות חובה:
1. לפתוח את האתר בעמוד #products.
2. לגלול לאזור הסיום.
3. לוודא שאין שם שדות טופס בכלל.
4. לוודא שרואים רק כותרת, טקסט, טלפון, מייל, כפתור וחתימה.
5. ללחוץ על “שליחת הזמנה”.
6. לוודא שהפעולה תקינה:
   - WhatsApp נפתח
   או
   - modal הזמנה נפתח
7. לוודא שה-orderModal עדיין עובד כשפותחים סל / הזמנה.
8. לוודא שהטופס המלא קיים רק בתוך modal ולא בתוך הדף.

חשוב מאוד:
לא להכניס את טופס ההזמנה לתוך contact-section.
לא להציג שדות קלט באזור הסיום.
לא למחוק את orderModal.
רק להחזיר הפרדה נכונה:
אזור צור קשר נקי בדף
טופס הזמנה מלא בתוך modal בלבד.
